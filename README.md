# Minishop 迷你购物车

一个用Django搭建的简单的购物小网站

所需Python包如下：

```
Django、mysqlclient、pymysql、pillow、pandas、xlwt
```

## 功能:

实现简单的商品选购与购物请求,拥有基本的注册页面与管理页面

简单的增删改查与用户数据导出功能
